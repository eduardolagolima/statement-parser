# statement-parser

CLI para extrair os proventos dos extratos da Avenue e B3.

### Como instalar?

```sh
npm install
npm i -g .
```

### Exemplos de como usar:

```sh
statement-parser --template=avenue examples/avenue.csv
```

```sh
statement-parser --template=b3 examples/b3.csv
```
